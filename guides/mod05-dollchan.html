Dollchan 8: Infinity

Installation Guide

  • Requires Dependencies found in "Essentials" at the top of the page.
  • (Microsoft Visual C++ 2017, DirectX and OpenAL).
  • Extract Archive, move game folder to the root of your drive.
  • Launch the mod using "Dollchan.exe".
  • (Optional) Switch game engine in "JSGME.exe" - Anomaly is recommended.
  • (Optional) Activate D-HUD in "JSGME.exe" to return some HUD indicators (radiation, damage etc.).
  • (Optional) Check the readme in the gamefiles for troubleshooting.
