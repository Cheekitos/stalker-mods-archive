Spatial Anomaly

Installation Guide

  • This mod is not a standalone and requires S.T.A.L.K.E.R.: Call of Pripyat (1.6.02)
  • There should be no gamedata or appdata folder in your main game folder (clean install)
  • Extract archive, copy content in your main game folder, replace files when prompted
  • Launch game with "Stalker-COP.exe"
  • (Optional) Move content of the "FOV 75" folder to your main game folder to change FOV
