Legend Returns

Installation Guide

  • Requires Visual C++ Redist 2013 x86, DirectX End-User Runtimes 2010.
  • Extract Archive, move game folder to the root of your drive.
  • Launch the mod using "Launch Legend Returns.bat".
  • (Optional) On first launch, reset weather & lighting settings to default.
  • (Optional) Open console with ~ and use "cam_inert 0.001" to fix headbob.
