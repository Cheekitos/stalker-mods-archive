Vector of Alienation

Installation Guide

  • Extract Archive, move game folder to the root of your drive.
  • Launch XR_3DA.exe in the bin folder.
  • (optional) Run JSGME.exe and add addons you want.
  • (optional) Add the Vector Weapon Pack.
