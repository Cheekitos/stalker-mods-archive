Incubator

Installation Guide

  • Extract Archive, then Unpack files by launching "INCUBATOR (23.03.2023).exe"
  • Copy the translation "gamedata" folder into your main game folder.
  • Launch the mod using "xrLauncher.exe" or "xrLauncher.bat".
  • (optional) Enable lightweight particles in JSGME for better performance.
