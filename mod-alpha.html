Mod Alpha - Installation Guide

Follow these steps to install Mod Alpha:

  1. Download the mod archive from the link provided.
  2. Extract it using WinRAR or 7-Zip.
  3. Copy the extracted files into your S.T.A.L.K.E.R. game directory.
  4. Overwrite existing files if prompted.
  5. Launch the game and enjoy.

Important: This mod is only compatible with version 1.0006.
